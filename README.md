# mastrobuoni.github.io

Academic website starter for GitHub Pages.

## Deploy

1. Create a GitHub repository named `mastrobuoni.github.io`.
2. In this folder, run:

```bash
git init
git add .
git commit -m "Initial academic website"
git branch -M main
git remote add origin https://github.com/<your-username>/mastrobuoni.github.io.git
git push -u origin main
```

3. In GitHub: Settings -> Pages.
4. Source: `Deploy from a branch`, Branch: `main`, Folder: `/ (root)`.
5. Your site will be live at `https://mastrobuoni.github.io/`.

## Customize

- Replace placeholder links in `index.html`.
- Add your PDF CV as `cv.pdf` in the root.
- Update contact links (Google Scholar, RePEc, etc.).
